Installing client dependencies:

1) `cd` into the `client` folder.
2) Run `npm install`.

Running the client in test mode:

1) `npm run start`

Installing server depencendies:

1) `cd` into the `server` folder.
2) `python -m pip install -U discord.py`

Running the server in test mode:

1) `python server.py`

Recommended tools:

1) https://sqlitebrowser.org/ - GUI for inspecting and editing the database.
2) https://git-scm.com/downloads - Linux-style shell with a bunch of useful commands.
3) https://nodejs.org/en/ - To run the client locally.
